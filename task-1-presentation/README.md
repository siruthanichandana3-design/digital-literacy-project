# Task 1 – Digital Literacy Awareness Infographic

## Tool Used
Canva (canva.com)

## About the Infographic

I created a one-page infographic titled "Digital Literacy for Students" 
with the tagline – Smart and Safe Use of Technology for Students.

It covers four topics:

- **What is Digital Literacy** – Using digital tools effectively and 
  communicating responsibly online.
- **Professional Online Presence** – Keeping profiles updated and 
  building a positive digital identity.
- **Safe Internet Practices** – Using strong passwords, avoiding 
  suspicious links and not sharing personal data.
- **Useful Digital Tools** – Google Docs, Canva and GitHub and 
  how students can use them.

The infographic ends with – *"Be Smart. Stay Safe. Stay Digital."*

## What I Found Interesting

Arranging information visually using Canva was more effective than 
plain text. The challenging part was deciding what to include while 
keeping the design clean and simple.
